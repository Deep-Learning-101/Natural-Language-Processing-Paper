# 🧠 DeepKE：開源中文知識圖譜抽取框架

知識圖譜 (Knowledge Graph) 的建立，極度仰賴精準的知識萃取工具。由浙江大學 NLP 實驗室 (ZJU-NLP) 牽頭開源的 **DeepKE (Deep Knowledge Extraction)**，是目前業界處理中文知識萃取非常成熟的深度學習工具包。

## 📄 學術文獻
[Ningyu Zhang, Xin Xu, Liankuan Tao, Haiyang Yu, Hongbin Ye, Shuofei Qiao, Xin Xie, Xiang Chen, Zhoubo Li, Lei Li, Xiaozhuan Liang, Yunzhi Yao, Shumin Deng, Peng Wang, Wen Zhang, Zhenru Zhang, Chuanqi Tan, Qiang Chen, Feiyu Xiong, Fei Huang, Guozhou Zheng, Huajun Chen, "DeepKE: A Deep Learning Based Knowledge Extraction Toolkit for Knowledge Base Population", arXiv preprint, arXiv:2201.03335, 2022](https://arxiv.org/abs/2201.03335)

## 🔗 開源資源與大模型進化

隨著大語言模型 (LLM) 的爆發，DeepKE 團隊也與時俱進，推出了基於 LLM 的智析抽取大模型 (DeepKE-LLM)：

* 💻 [**DeepKE GitHub 官方倉庫**](https://github.com/zjunlp/DeepKE/blob/main/README_CN.md)：基於深度學習的開源中文知識圖譜抽取框架。
* 📰 [**版本更新介紹**](https://mp.weixin.qq.com/s/PL_21wmRqmAPr0i34yVEeA)：開源知識圖譜抽取工具正式發布大模型版 DeepKE-LLM。
* 🤖 [**ModelScope 魔搭社區**](https://modelscope.cn/models/ZJUNLP/DeepKE-LLM/summary)：KnowLM 系列之智析抽取大模型下載與體驗。
